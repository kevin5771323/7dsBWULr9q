# 停车场管理系统介绍 java1117

## 项目概述

停车场管理系统是一个基于B/S架构和MVC模式的Web应用程序，采用前后端分离的设计，前端使用Vue框架，后端采用Java技术栈，以SpringBoot框架为基础进行开发。

## 技术栈

- **架构**: B/S、MVC
- **开发语言**: Java
- **数据库**: Mysql
- **后端框架**: SpringBoot
- **持久层框架**: Mybatis-Plus
- **前端框架**: Vue

## 系统环境与开发工具

- **系统环境**: Windows
- **开发环境**: IDEA、JDK1.8、Maven、Mysql8.0

## 功能模块

### 用户角色

#### 普通用户

- 个人信息管理
- 预定停车位
- 缴费信息管理

#### 管理员用户

- 用户信息管理
- 车辆信息管理
- 停泊车辆管理
- 车辆进出管理

## 运行环境

- **服务器**: 推荐使用与开发环境一致的Windows服务器，或其他支持Java应用的服务器环境
- **数据库**: Mysql 8.0或以上版本
- **浏览器**: 现代浏览器，推荐使用Chrome、Firefox等
